# Sherlock Analyzer

Sherlock Analyzer is a multi-agent AI reasoning demo created for the Complete.dev AI Hackathon.

The system analyzes real-world events and evaluates whether public explanations are supported by documented evidence.

## Live Demo

https://tw6aktoe.run.complete.dev

## What the project demonstrates

Instead of generating a narrative answer like a chatbot, Sherlock Analyzer performs structured reasoning:

1. Defines the investigation scope
2. Builds a verified timeline of events
3. Classifies public statements
4. Detects contradictions between claims and evidence
5. Produces a confidence-rated explanation

The goal of the project is to demonstrate auditable and transparent AI reasoning.

## Example Case

Volkswagen Dieselgate (2015)

The system automatically identifies discrepancies between corporate public statements and regulatory investigation records.
